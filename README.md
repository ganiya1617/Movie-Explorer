# Movie Explorer
 
A web application for discovering and exploring movies using the TMDB (The Movie Database) API. Browse trending films, search by title, filter by genre, sort by rating or year, and save your favourites.
 
---
 
## Project Purpose
 
Movie Explorer is built as part of an individual project to demonstrate skills in JavaScript, public API integration, and UI development. The app fetches real time movie data and lets users interact with it through search, filtering, and sorting  all powered by JavaScript Array Higher-Order Functions.
